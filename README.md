### What I'm Building

<table>
<tr>
<td width="50%" valign="top">

#### [kost](https://github.com/yashtandon019/kost)
**Kubernetes cost anomaly detector**

A lightweight operator that watches cluster resource usage, learns normal spending patterns via statistical baselines (Welford's algorithm), and alerts when costs deviate — with human-readable explanations powered by pluggable LLM backends.

#### [karpenter-lens](https://github.com/yashtandon019/karpenter-lens)
**Karpenter node efficiency analyzer**

CLI tool that scores how well Karpenter-managed nodes are packed, detects overprovisioned instances, calculates DaemonSet overhead tax, and simulates bin-packing what-if scenarios with cost comparisons.
